Developer Guide

Welcome! Follow these steps to set up MondayMoney for development.

Setup

git clone https://github.com/thiagocolen/monday-money.git
cd monday-money
npm install
npm run data-reset
npm run dev

Testing

We use Playwright for E2E testing. Run the full suite with:

npm run test

Or run a specific test file:

npm run test:file tests/categories-tags.spec.ts

Building

To create a portable build for Windows:

npm run build

The output will be in release/${version}/.
