Architecture Overview

MondayMoney is built with modern web technologies, packaged as a desktop application using Electron.

Tech Stack

  • Runtime: Electron - Provides the desktop environment and file system access.
  • Frontend: React 19 - UI library for building the interface.
  • Bundler: Vite - Fast development server and build tool.
  • Styling: Tailwind CSS 4 - Utility-first CSS framework.
  • UI Components: Radix UI & Shadcn/UI - Accessible UI primitives.

Data Storage Strategy

We use a File-based Storage approach. Instead of a traditional database, we store everything in CSV files within the core/data/ directory. This ensures your data is transparent, portable, and easy to backup.

Maintenance Tools

The project includes specialized scripts to maintain data health:

  • npm run integrity-check: Validates CSV consistency.
  • npm run data-reset: Resets data with fresh seed transactions (for dev).
  • npm run protect-files: Sets data files to read-only at the OS level.
Data belongs to the universe, but your CSVs belong to you.
