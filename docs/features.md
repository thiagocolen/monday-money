# Features

MondayMoney provides a comprehensive set of features to manage your personal finances without compromising your privacy.

## 📊 Data Visualization
Interactive charts powered by **Recharts** allow you to understand your spending habits at a glance. You can view:
- Monthly spending trends.
- Category breakdowns.
- Yearly comparisons.

## 📂 CSV Integration
Easily import and manage transaction history using standard formats. We support various bank formats and provide tools to map them to the MondayMoney ledger.

## 🏷️ Category & Tag Management
Organize your transactions with customizable categories and tags.
- **Auto-categorization:** Smart rules to categorize recurring transactions.
- **Bulk Tagging:** Tag multiple transactions at once to save time.

## 🔒 Privacy & Security
- **Local Storage:** All data is stored in `core/data/` as CSV files.
- **No External Dependencies:** The app doesn't call home or sync with external servers.
- **File Protection:** Built-in scripts to protect your data files from accidental modification.

## 🛠️ Integrity Checks
Ensure your ledger remains consistent with built-in validation tools that check for duplicate transactions or data corruption.
