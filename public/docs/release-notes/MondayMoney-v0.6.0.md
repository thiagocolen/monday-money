# Release Notes - MondayMoney v0.6.0

**Version:** 0.6.0
**Date:** 2026-05-05
**Author:** MondayMoney Assistant

## Description
This major update (v0.6.0) introduces comprehensive investment tracking and advanced data management capabilities. We have also modernized our tech stack to leverage the latest features in React 19 and Tailwind CSS 4, ensuring a faster and more maintainable application.

## Feature List

### 📈 Investment Tracking (Binance)
*   **New Investments Page:** A dedicated workspace to monitor your Binance exchange activity.
*   **Transaction History:** View all your exchange operations in a searchable, paginated table.
*   **Asset Flow Monitoring:** Specialized tabs for tracking Crypto and Fiat deposits/withdrawals.
*   **Robust Data Filtering:** Built-in logic to exclude internal seed and chain transactions, focusing on your actual movements.

### 💾 Data Management & Backup
*   **New Backup Page:** A centralized hub for managing your application data and settings.
*   **Full System Export:** Create complete `.zip` archives of your entire ledger and configuration with a single click.
*   **Custom Export Path:** Configure a dedicated folder on your machine for storing your backups.
*   **Application Reset:** New "Danger Zone" feature to safely clear all data when starting fresh.

### ⚡ Tech Stack Upgrade
*   **React 19:** Leveraged the latest React features for improved performance and developer experience.
*   **Tailwind CSS 4:** Upgraded to the newest version of the utility-first CSS framework for enhanced styling capabilities.
*   **Vite 8:** Powered by the latest version of the Vite build tool.
*   **Lucide React & Phosphor Icons:** Integrated modern icon sets for a cleaner and more consistent UI.

### 🛠️ UI/UX Enhancements
*   **Enhanced Navigation:** Refined layout with active path indicators and improved keyboard shortcuts.
*   **Global Drag & Drop Protection:** Improved Electron event handling to prevent accidental file navigation.
*   **Improved Tooltips:** Added comprehensive keyboard shortcut documentation within the UI.
*   **Theme Toggle:** Integrated theme switching directly into the main header.

### 🔍 Project Maintenance
*   **Documentation Update:** Comprehensive update of the project's documentation website to reflect all new features and technical changes.
*   **Integrity & Security:** Enhanced data protection scripts and integrity checks for the local CSV ledger.
